# Portal Imperial — Sistema POS

Sistema de punto de venta web para restaurante. Funciona en Windows, Android, iPhone y tablets.

## Funciones
- Login por roles (Administrador, Cajero, Supervisor, Cocina)
- Ventas tipo Mesa / Para Llevar / Domicilio
- Pantalla de cocina con temporizador y sonido
- Caja con gastos, nómina y cierre por empleado
- Domiciliarios, clientes frecuentes, reportes y auditoría
- Numeración consecutiva de facturas (PI-000001...)
